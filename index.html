Teknologi yang Digunakan

Di kelas ini, kamu akan membuat aplikasi web sendiri menggunakan teknologi dibawah ini.
HTML & CSS

Pelajari teknologi utama dalam pembuatan website. HTML menyediakan struktur halaman sedangkan CSS mengatur layout dan tampilan web.

Javascript

JavaScript adalah bahasa pemrograman tingkat tinggi yang merupakan salah satu bahasa pemrograman terpopuler di kalangan programmer.

jQuery

jQuery adalah sebuah library JavaScript open souce yang dirancang untuk mempermudah penggunaan JavaScript dalam sebuah website.

GitHub

Dengan GitHub, kamu dapat memasukkan source code dari sebuah project dan mengerjakannya dengan progammer lain pada saat bersamaan.
